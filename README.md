# Aura Landing

Welcome to Aura Landing! This is a Next.js starter project designed to provide a visually appealing, high-performance landing page. It comes pre-configured with a modern tech stack to help you launch your next idea faster.

This project was generated by an AI assistant and includes a full landing page, database setup with Prisma, and CI/CD readiness.

## Core Features

- **Modern Frontend**: Built with **Next.js 15** (App Router), **React**, and **TypeScript**.
- **Stunning UI**: Styled with **Tailwind CSS** and **Shadcn UI** components for a clean, professional look.
- **Database Ready**: **Prisma ORM** pre-configured for a **PostgreSQL** database. Includes a waitlist feature out of the box.
- **CI/CD Prepared**: Scripts for linting (`ESLint`), type-checking (`TypeScript`), and building, ready for deployment with **Vercel** and **GitHub Actions**.
- **Responsive Design**: Looks great on all devices, from mobile phones to desktops.

## Getting Started

Follow these steps to get your local development environment up and running.

### 1. Clone the Repository

First, get a copy of the project on your local machine.

```bash
git clone <your-repository-url>
cd aura-landing
```

### 2. Install Dependencies

Install all the necessary packages using npm.

```bash
npm install
```

### 3. Set Up Environment Variables

This project uses Prisma to connect to a PostgreSQL database. You'll need to set up your database connection string.

1.  Copy the example environment file:
    ```bash
    cp .env.example .env
    ```
2.  Open the `.env` file and replace the placeholder `DATABASE_URL` with your actual PostgreSQL connection string.

    ```env
    DATABASE_URL="postgresql://user:password@localhost:5432/mydatabase?schema=public"
    ```

### 4. Apply Database Schema

Push the pre-defined Prisma schema to your database. This will create the `WaitlistEntry` table.

```bash
npx prisma db push
```

### 5. Run the Development Server

You're all set! Start the Next.js development server.

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) in your browser to see the result.

## Available Scripts

- `npm run dev`: Starts the development server.
- `npm run build`: Creates a production-ready build of the application. This also runs `prisma generate`.
- `npm run start`: Starts the production server.
- `npm run lint`: Lints the codebase using ESLint.
- `npm run typecheck`: Checks for TypeScript errors.

## Deployment

This project is optimized for deployment on [Vercel](https://vercel.com/), the platform from the creators of Next.js.

### Automated Deployment with Vercel and GitHub

1.  **Push to GitHub**: Create a new repository on GitHub and push your local code to it.
2.  **Import Project on Vercel**: Log in to your Vercel account and import the GitHub repository.
3.  **Configure Environment Variables**: During the import process, Vercel will prompt you to add environment variables. Add your `DATABASE_URL`.
4.  **Deploy**: Click "Deploy". Vercel will automatically build and deploy your application. Subsequent pushes to the main branch will trigger automatic deployments.

Vercel's integration with Next.js is seamless and requires zero configuration.
